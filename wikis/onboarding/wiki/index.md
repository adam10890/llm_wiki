# Onboarding Wiki — Index

This wiki is the **first place an agent looks** when it needs to use the SharedBrain (the LLM Wiki long-term memory system). It is small on purpose: every page here is operating instructions, not domain knowledge.

## Reading order

1. **`_start_here.md`** — how the wiki layer works, when to read vs. write, how to coordinate with Pen & Paper. Read first.
2. Pick a page below based on what you are trying to do.

## Catalog

| Path | When to read |
|---|---|
| `_start_here.md` | First time, or whenever re-orienting. Operating rules. |
| `index.md` | This page — to pick the next page. |
| `log.md` | Recent operations against this wiki; append-only audit trail. |

## What this wiki is *not*

- Not a place to dump domain knowledge — that goes in `commons`, `general`, `about_user`, or a project wiki.
- Not a scratchpad — that is the Pen & Paper plugin (`a0_pen_paper`).

## Bridge to Pen & Paper

- **Active drafting / thinking** → `pen_paper` tool. See `a0_pen_paper/data/templates/_start_here.md` for its operating rules.
- **Persistent knowledge that compounds across sessions** → SharedBrain wikis. Start with `_start_here.md` in this wiki.

Hard ceiling for every page in this vault: **500 lines**. Target: ≤400.
